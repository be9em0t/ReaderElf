Overview

What ReaderElf does:
- personal text reader, no requirement to be deployed in bulk.
- reads aloud ebooks, web pages, blogs or any text pasted or provided as a .txt file.
- free or low-cost implementation
- reader quality at the same level as Copilot or Grok implementation - good understanding of intonation and context. NO DEFAULT ROBOTIC TTS.

Basic workflow

1. Open a file or paste text
2. Choose a voice and speed
3. Start/Pause/Stop playback

Advanced options

- Save reading position
- Export audio
- Keyboard shortcuts

Troubleshooting

- No sound: check system volume and permissions
- No voice options: ensure TTS service configured

Contact & feedback

Please file issues in the repository or contact the maintainers.
