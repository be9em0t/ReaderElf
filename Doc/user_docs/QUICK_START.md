# Quick Start — ReaderElf

This quick start helps a reader begin using ReaderElf to read aloud text.

1. Install the app (or run locally) — instructions will be added to `Doc/user_docs/INSTALL.md`.
2. Open a document or paste a URL into the app.
3. Press Play to start text-to-speech. Use the playback controls to pause, skip or change speed.

Key features (planned):
- Multiple voice selections
- Adjustable speed and pitch
- Save/restore reading positions across devices

For more detailed instructions, see `USER_GUIDE.md`.
