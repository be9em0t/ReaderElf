# Documentation for ReaderElf

This folder stores product requirements, user-facing documentation, templates and meeting notes for the ReaderElf project.

Structure

- `requirements/` - requirements files and feature specs
- `user_docs/` - guides, quick start, FAQ for end users
- `templates/` - PRD, roadmap and other authoring templates
- `notes/` - meeting notes and research

Start here:

- `user_docs/QUICK_START.md` - quick start for users
- `requirements/REQUIREMENTS_AZURE_VERSION.md` and `REQUIREMENTS_BROWSER_VERSION.md` - requirements already present
